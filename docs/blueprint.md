# **App Name**: YourGroceriesUSA Storefront

## Core Features:

- Product Catalog & Categories: Display a browseable catalog of products on the homepage, organized into categories such as Makeup, Skincare, Haircare, Vitamins, Bags, and Watches. Each product will show an image, name, and pricing.
- Product Detail Pages: Dedicated pages for each product, presenting comprehensive descriptions, high-resolution images, and all relevant product-specific information.
- WhatsApp Order Integration: Implement a 'WhatsApp to Order' button on each product page, allowing customers to easily initiate an order directly via WhatsApp, streamlining the purchasing process.
- Admin Order Viewer: A secured administrative route (/admin) where the client can view and manage incoming orders from the e-commerce platform.
- Firebase Backend Integration: Utilize Firebase for robust backend services, including secure storage and retrieval of product data, category listings, and customer order information.
- AI Product Description Tool: An AI-powered tool within the admin panel to generate compelling and optimized product descriptions based on input features or keywords, facilitating efficient content creation for new listings.

## Style Guidelines:

- Primary color: A sophisticated Rich Rose (#B2146B) to denote key interactive elements, important headings, and brand highlights, adding depth to the beauty store aesthetic.
- Main background: A clean and expansive White (#FFFFFF) to provide a modern, minimalist foundation for product display and content, as requested.
- Secondary background: A delicate Off-White with a subtle rose tint (#FAF5F8) for subtle content separation, card backgrounds, or elegant borders, harmonizing with the overall palette.
- Accent color: The specified Soft Pink (#FFC0CB), used for gentle decorative elements, less prominent text, and tender visual cues, enhancing the elegant and welcoming feel.
- Emphasis accent: A vibrant Magenta (#E500E5) for critical call-to-action buttons, active states, and high-impact messaging, providing a strong visual contrast and engaging interaction.
- All text will use 'Inter' (sans-serif), chosen for its modern, clean lines and excellent readability across various text sizes, ensuring a professional and accessible user experience.
- Employ a minimalist set of line-art icons. Icons will be subtly styled to align with beauty, wellness, and essential goods, contributing to a refined and contemporary visual language.
- The layout will be clean, utilizing a grid-based system for product listings and category displays, with generous whitespace to enhance focus on individual items and improve visual comfort. It will be fully responsive for all device types.
- Introduce subtle and smooth micro-animations for elements such as product hover states, button feedback, and page transitions, contributing to a premium, polished, and intuitive user interface.
